## DSA Search Engine

Deployed Project | Feb 2025 – Mar 2025
Live: https://dsa-based-search-engine-mongodb-5.onrender.com/
A full-stack web application that allows users to search for Data Structures and Algorithms (DSA) problems with high relevance. The search engine uses the TF-IDF algorithm to rank problems based on user queries and provides the TF-IDF score along with each result.

Features

Accurate Search Results
Implemented the TF-IDF (Term Frequency–Inverse Document Frequency) algorithm to retrieve and rank problems according to query relevance.

TF-IDF Score Display
Shows the computed TF-IDF score for each problem to give users an understanding of the relevance of each result.

Web Scraping
Scraped over 900 problems from popular coding platforms: LeetCode, CSES, CodeChef using Selenium and Beautiful Soup.

## Full-Stack Web Application
Built using:

Frontend: HTML, CSS, JavaScript

Backend: Node.js, Express.js

Database: MongoDB Atlas hosted on Azure

## Cloud Deployment
Deployed the application on Render for easy access and scalability.

## Results Display

The frontend shows relevant problems along with their TF-IDF scores to help users identify the most suitable problems.
